# Find buffer width that produces target area per side

Find buffer width that produces target area per side

## Usage

``` r
find_border_width(border_geom, target_area, mask_geom, crs)
```

## Arguments

- border_geom:

  Border line geometry (sfc)

- target_area:

  Target area for each side

- mask_geom:

  Optional mask geometry

- crs:

  CRS of the geometries

## Value

Buffer width
